## Getting Started

First, run the development server:

1. Install Packages

```bash
npm install
# or
yarn install
```

2. Run app

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Pages

- [Login Page](http://localhost:3000/login)
- [Customers](http://localhost:3000)
